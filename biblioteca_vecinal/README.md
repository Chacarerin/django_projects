# Biblioteca Vecinal

Sistema web diseñado para la administración y control de una biblioteca de alcance comunitario. El objetivo del proyecto es digitalizar el inventario de libros y facilitar el proceso de préstamos.

## Características Técnicas
- **Modelos:** Gestión de Entidades como Libro, Autor, Socio y Préstamo.
- **Relaciones:** Implementación de claves foráneas (ForeignKey) para vincular los préstamos con los libros y los usuarios.
- **Vistas:** Vistas basadas en funciones (FBV) para listar el catálogo y registrar devoluciones.
- **Panel de Administración:** Uso del `admin` de Django para facilitar la carga inicial de datos por parte de los administradores.
