# Práctica Final ORM

Aplicación integradora diseñada exclusivamente para probar y aplicar conocimientos profundos del Object-Relational Mapping (ORM) de Django, separando la lógica de base de datos de las vistas web tradicionales.

## Características Técnicas
- **Relaciones Complejas:** Implementación y consulta a través de relaciones `OneToOne`, `ForeignKey` y `ManyToMany`.
- **Agregaciones y Anotaciones:** Uso de las funciones `Count`, `Sum`, `Avg` sobre conjuntos de consultas (QuerySets) para obtener reportes analíticos.
- **Consultas Q (Q Objects):** Construcción de consultas dinámicas con operadores lógicos OR/AND complejos.
