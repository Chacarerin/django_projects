Este proyecto marca la transición estructural desde aplicaciones monolíticas renderizadas por el servidor, hacia arquitecturas desacopladas mediante la exposición de servicios RESTful (Representational State Transfer).

## 📌 Contenidos Principales
- **Serialización (Serializers):** Desarrollo de clases especializadas que asumen la responsabilidad bidireccional de convertir objetos de Python en diccionarios JSON (para lectura) y viceversa (para creación/actualización con validación).
- **Vistas API (APIViews & ViewSets):** Reemplazo de las vistas HTML tradicionales por controladores orientados exclusivamente al intercambio de flujos de datos.
- **Enrutamiento Dedicado (Routers):** Generación automática de las convenciones de URLs requeridas para un servicio CRUD completo sobre entidades de películas.

## ⚙️ Tecnologías y Frameworks Aplicados
- **Django REST Framework (DRF):** Una potente extensión sobre la base de Django que provee las piezas faltantes (Serializers, ViewSets, Token Authentication) requeridas para la construcción de Web APIs robustas y escalables con estándares modernos de la industria.
