# Movies API

Este proyecto representa un salto hacia la arquitectura desacoplada, utilizando **Django REST Framework (DRF)** para construir una interfaz de programación de aplicaciones (API) RESTful.

## Características Técnicas
- **Serializadores (Serializers):** Conversión bidireccional entre los objetos complejos del ORM de Django y tipos de datos nativos de Python (renderizados finalmente en JSON).
- **Vistas Basadas en Clases (APIViews):** Implementación de endpoints estándar para operaciones CRUD (GET, POST, PUT, DELETE).
- **Enrutamiento (Routers):** Configuración automática de URLs para las vistas de la API, facilitando el mantenimiento y escalabilidad.
