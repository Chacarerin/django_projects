# 🍎 Sistema Frutería

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![Django](https://img.shields.io/badge/django-%23092E20.svg?style=for-the-badge&logo=django&logoColor=white)

Aplicación enfocada en la gestión de logística inversa y control operativo de un inventario perecedero a nivel comercial básico.

## 📌 Contenidos Principales
- **Mapeo de Atributos Críticos:** Declaración de estructuras de persistencia considerando la variabilidad escalar del negocio (precios unitarios versus peso).
- **Navegabilidad List-to-Detail:** Construcción de un flujo de usuario donde se prioriza la extracción de listados holísticos (vistas maestras) seguida por un acceso minucioso a características detalladas (vista de detalle).
- **Procesamiento de Visualización:** Implementación de capas de presentación modulares (Templates) priorizando legibilidad y minimizando lógica algorítmica pesada en la interfaz final.

## ⚙️ Tecnologías y Frameworks Aplicados
- **SQLite3 y Django ORM:** Utilizados primariamente por su inmediatez en ambientes de desarrollo y pruebas, la combinación de Django con la base relacional embebida SQLite3 garantiza transaccionalidad total (ACID) con configuraciones sumamente ligeras.
