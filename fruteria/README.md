# Sistema Frutería

Aplicación diseñada para la gestión operativa de un local de venta de frutas y verduras, controlando el stock y los precios.

## Características Técnicas
- **Modelado de Inventario:** Representación de productos perecederos, considerando precios por unidad o peso.
- **Vistas List/Detail:** Diferenciación entre la vista que muestra el catálogo completo y la que muestra el detalle particular de un producto.
- **Integración Frontend:** Presentación de los datos mediante plantillas limpias y optimizadas para facilitar la navegación del usuario final.
