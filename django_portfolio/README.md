Desarrollo de una *Landing Page* dinámica concebida para la exhibición formal de trayectoria y acervo profesional. La distinción académica de este modelo radica en la integración paralela de recursos mediáticos estáticos y dinámicos (carga de archivos).

## 📌 Contenidos Principales
- **Manejo de Carga Binaria (Media Files):** Estudio de las variables de entorno `MEDIA_URL` y `MEDIA_ROOT` requeridas para procesar el almacenamiento asíncrono de imágenes referenciales y vinculaciones en bases de datos.
- **Adaptación Estructural del Panel Base:** Intervención sobre las clases del archivo `admin.py` para adecuar la carga, edición y previsualización de elementos mediáticos directamente desde el administrador sin desarrollo frontend adicional.
- **Arquitectura de Ficheros Estáticos:** Metodología estricta para la integración de dependencias CSS/JS personalizadas bajo el directorio normativo `static/`.

## ⚙️ Tecnologías y Frameworks Aplicados
- **Django Template Engine:** Este motor interno realiza el análisis sintáctico de etiquetas y variables permitiendo concatenar bucles sobre listas dinámicas extraídas desde la base de datos directamente en el código de marcaje (HTML) que el navegador requiere.
