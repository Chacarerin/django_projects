# Portafolio Profesional

Aplicación orientada a la presentación de currículums y trabajos destacados, ideal para profesionales independientes que deseen administrar su propia página de aterrizaje (Landing Page).

## Características Técnicas
- **Gestión de Medios (Media Files):** Configuración de `MEDIA_URL` y `MEDIA_ROOT` para permitir la carga y visualización de imágenes representativas de los proyectos.
- **Panel Administrativo Personalizado:** Adaptación del panel de Django para administrar fácilmente las entradas del portafolio.
- **Diseño Estático:** Integración de archivos CSS y JavaScript propios en el directorio `static/` para un estilo visual único.
