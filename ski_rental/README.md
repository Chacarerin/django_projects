# Ski Rental

Sistema de administración orientado a un centro de esquí para la gestión del arriendo de su equipo deportivo.

## Características Técnicas
- **Modelado de Dominio:** Representación de inventario especializado (esquís, botas, bastones) y la lógica de negocio temporal (fechas de inicio y fin del arriendo).
- **Validaciones de Negocio:** Aseguramiento de la coherencia en los registros (evitando arriendo de equipo inhabilitado o ya arrendado).
- **Vistas Especializadas:** Listados de disponibilidad en tiempo real y paneles de control para el administrador del recinto.
