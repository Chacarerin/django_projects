# Django Blog

Plataforma de blog completa que permite a los usuarios publicar, leer y gestionar artículos de contenido. Es una demostración clásica de las capacidades de contenido y autenticación de Django.

## Características Técnicas
- **Autenticación:** Uso del sistema de usuarios de Django para diferenciar entre visitantes anónimos y autores registrados.
- **Gestión de Contenido:** Operaciones CRUD completas sobre el modelo "Post" o "Artículo".
- **Sistema de Plantillas:** Uso intensivo de herencia de templates (`{% extends %}`) para mantener un diseño consistente.
- **Formularios:** Implementación de `ModelForms` para simplificar la captura y validación de datos ingresados por el usuario.
