# Django CRUD Básico

Este proyecto es de carácter formativo y se centra estrictamente en dominar las cuatro operaciones fundamentales sobre bases de datos: Crear, Leer, Actualizar y Eliminar (CRUD).

## Características Técnicas
- **Abstracción de Base de Datos:** Demostración de cómo el ORM interactúa con la base de datos sin requerir SQL directo.
- **Rutas (URLs):** Parametrización de URLs (ej. `/<id>/update`) para acceder a registros específicos.
- **Protección CSRF:** Implementación de tokens de seguridad en todos los formularios para prevenir ataques Cross-Site Request Forgery.
- **Redirecciones:** Flujo lógico en las vistas tras enviar un formulario exitosamente.
