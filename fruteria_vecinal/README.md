# Frutería Vecinal

Versión extendida y mejorada del sistema de frutería básica, incorporando funcionalidades adicionales para cubrir las necesidades de un comercio a escala de barrio.

## Características Técnicas
- **Autenticación Basada en Roles:** Diferenciación entre usuarios clientes (que ven catálogo) y usuarios vendedores (que administran stock).
- **Procesamiento de Formularios:** Uso avanzado de formularios para actualizar múltiples registros simultáneamente.
- **Consultas con el ORM:** Filtros avanzados para categorizar productos según su tipo o disponibilidad temporal.
